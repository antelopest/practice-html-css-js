# UI KIT

## Badge HTML Element

Badge HTML element to display the current status order.

| Status    | `background` | `color` |
|-----------|--------------|---------|
| `success` | `green`      | `white` |
| `error`   | `red`        | `white` |
| `warning` | `orange`     | `black` |
| 'other'   | `gray`       | `white` |
